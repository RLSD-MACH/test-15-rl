Purchase Delivery Order Split

This module allows you to split the purchase delivery order according to the product's availability in the warehouse

Few Words About The Purchase Delivery Order Split:

Purchase Delivery Order Split module provides an easy interface for managing the moving lines inside the warehouse. Sometimes due to unavailability of products, providing the delivery of order on the committed time become a tedious issue in such case the warehouse manager can use this plug-in to make the order Purchase delivery in multiple batches.

Work Flow: How it Works
Purchase Order Creation
  • Create the purchase order
  • Confirm the purchase order
Delivery Order Generated
  • After confirmation of the purchase order, the delivery order is generated
  • Click the smart button to jump to the delivery order
Delivery Order View
  • Click the Split button to split the delivery order
Update Quantity
  • Here update the available quantity in the warehouse you want to transfer
New Splitted Delivery Orders
  • After split the new delivery orders are generated
  • The old delivery order is canceled
