VIES

Users can request a validation of a contacts VAT-number from VIES. They just need to click the little "eye" next to the VAT-number.

Users can see the response from VIES

User can also see the send request.

Users can view the history of validations and access them - which is needed if sales have been made without VAT within EU.

A quick action to this view is available on the contact form.

It will appear on the contact formview with a red banner, if the VAT-validation response is false.

It will appear on the sales order formview with a red banner, if the VAT-validation response is false.

It will appear on the sales order formview with a red banner, if the VAT-validation response is false.

It will only appear if the sales order is in draft.

It will appear on the account moves formview with a red banner, if the VAT-validation response is false.

It will only appear if the move is in draft.

The module requires some information in order to work. They can be filled out in the settings menu.
