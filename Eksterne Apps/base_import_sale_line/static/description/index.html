Features

Import lines from CSV / Excel file

You can easily import your sale order lines from data in an excel or csv file. Each column will be recognized and mapped to a field of sale order line.

Test data before importing

You can test your data and fix them before importing (if needed). It will check your data line by line, show error message at specific line number (if any).

Download a template file

This module gives you a template which you can download, update your data and import. You can add more columns if you want.

Mapping fields easily

If the header in your file is not good enough to be recognized automatically, you can re-choose a corresponding field by yourself, directly in the interface.
